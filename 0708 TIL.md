#### Frontend
- 서비스의 앞부분을 만드는 분야
- html, css, javascript를 통해 화면을 구성하고 동작
- 프론트엔드와 웹디자인은 다른 영역이나, 프렌으로 통합하는 추세

#### Backend
- 서비스의 뒷부분을 만드는 분야/ server sided
- java, python, ruby 
- spring, Django, Ruby on Rails 등의 역량 필요

#### DevOps (데브옵스)
- 요즘 핫한 분야
- 서버 자체를 하나의 프로그램처럼 다루는 것
- 개발(Dev)+운영(Ops)
- 배포, 스케일 인/아웃, 모니터링 등의 인프라 분야
- docker, kubernetes jenkins 등의 역량 필요

#### Data Scientist (데이터 사이언티스트)
- 쌓여 있는 데이터를 의미 있는 데이터로 도출하는 분야
- vs. 데이터 엔지니어: 데이터의 수집/가공/저장이 주 업무
- 데이터 전처리, 모델 학습, ML 파이프라인 구축 등을 담당



### 공고 읽는 법

- RnR

  - Role (역할)

  - Responsibilities(책임) :뭘 하게 될지

  - requirements(요구사항): 뭘 할 줄 알아야 하는지

  - Good to have(우대사항)

    

- 컬쳐핏 면접: 해당 회사가 추구하는 분위기에 맞는지 

  - ex 배달의 민족: 보통의 사람들이 모여 비범한 결과를 내는 팀

- 자격요건에 부합하지 않더라도, 일단 서류는 지원해보자.
